# bot-basics-train-gate
Open source train gate using Barnabas Robotics Bot Basics Kit

Teacher (Kent Kaiser) created this as a remix of the Barnabas bot basics project for an afterschool program.

Remix additions include:
- 3D printed enclosure
- IR sensor to sense the train

Video of gate in action: https://www.youtube.com/watch?v=bWuli3N5wUo

Materials Used:
- 3D prints (CAD files available here)
- Bot Basics Kit: https://shop.barnabasrobotics.com/collections/robots-and-coding/products/bot-basics-kit-intro-to-arduino
- IR Sensor Module: https://shop.barnabasrobotics.com/products/ir-sensor-module?_pos=1&_sid=9d1339bc1&_ss=r
